# Open on Spotify

Browser extension that allows you to open URLs in Spotify desktop client with a click.

## Attributing

Icons made by https://iconmonstr.com/

## Install

    npm install

## Development

    npm run dev chrome
    npm run dev firefox
    npm run dev opera
    npm run dev edge

## Build

    npm run build chrome
    npm run build firefox
    npm run build opera
    npm run build edge
